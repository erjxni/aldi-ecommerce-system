# Legacy Directory

This directory serves as an archive for legacy code, prototypes, and old configurations that are no longer actively used by the main system, but are kept for historical reference.

## Files
- `app.py`: The original Flask-based prototype for the user login story.
- `server.js`: An old Express.js backend prototype used before migrating to FastAPI.
- `package.json` & `package-lock.json`: The NPM dependencies for the old Express server.
- `requirements.txt`: The Python dependencies required for the old Flask prototype.

## Directories
- `static/`: Static assets (CSS, JS) used specifically by the Flask prototype.
- `templates/`: HTML Jinja templates used by the Flask prototype.
