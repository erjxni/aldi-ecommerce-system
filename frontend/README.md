# Frontend Directory

This directory contains the Vite-powered Vanilla JS, HTML, and CSS application.

## Files
- `index.html`: The main landing page of the e-commerce store.
- `login.html`: The user authentication and registration page.
- `products.html`: The main catalog page displaying all available products.
- `product-detail.html`: A page dedicated to displaying detailed information for a single product.
- `admin.html`: The administrator dashboard for viewing sales losses and customer metrics.
- `package.json` & `package-lock.json`: NPM configuration files defining project metadata, build scripts, and dependencies (like Vite).
- `.gitignore`: Specifies which frontend files (like `node_modules`) should be ignored by Git.

## Directories
- `src/`: Contains the main Vanilla JS source files implementing frontend logic (e.g., API calls, cart management, authentication state).
- `assets/`, `css/`, `js/`: Directories containing styles, static assets, and compiled or vendor scripts.
- `public/`: Static assets that are served directly without processing by Vite.
- `node_modules/`: Automatically generated folder containing installed NPM packages.
